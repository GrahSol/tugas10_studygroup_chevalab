# Week 10 - ExpressJS Separate Routes, Controllers, Middleware

## Deskripsi

Project ini merupakan implementasi CRUD API sederhana menggunakan ExpressJS dengan konsep pemisahan:

* Routes
* Controllers
* Middleware

---

# Struktur Folder

```txt
tugas10
├── node_modules
├── package.json
├── package-lock.json
└── src
    ├── app.js
    ├── routes
    │   └── v1.js
    └── controllers
        └── todo.controller.js
```

---

# Install Dependencies

```bash
npm install
```

---

# Menjalankan Project

```bash
npm start
```
